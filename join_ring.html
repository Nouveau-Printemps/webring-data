Rejoindre le webring

Pour rejoindre le webring, vous devez faire une pull request sur GitHub pour expliquer vos motivations.

Nous acceptons tout le monde, y compris si vous n'êtes pas membre du Nouveau Printemps. L'unique condition est d'être en accord avec nos valeurs.

Vous pouvez aussi créé votre propre webring en utilisant notre logiciel : Webring. Il est entièrement gratuit, open-source et customisable.
